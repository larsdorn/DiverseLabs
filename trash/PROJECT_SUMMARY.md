### Parallax Scrolling

Uses CSS Grid and Flexbox. Automatically adapts to:
- Desktop (1024px+)
- Tablet (768px-1024px)

- Auto-plays, muted, looping
- Responsive sizing
- Fallback to placeholder image

### Adding Your Video
1. Create 8-15 second video at 1920×1080
2. Compress to < 5MB
3. Save to `assets/videos/hero-background.mp4`
4. Update source in `index.html` line 60

## 🎨 Color Customization

Easy to change main colors. Edit `styles.css` line 10-20:

```css
:root {
    --neon-cyan: #00ff88;    /* Change this */
    --neon-pink: #ff006e;    /* And this */
    --primary-black: #0a0a0a; /* And this */
}
```

Try these combinations:
- 💚 Green/Orange: `#00ff88` / `#ff6600`
- 💜 Purple/Gold: `#b74fff` / `#ffbe0b`
- 💙 Blue/Yellow: `#00d4ff` / `#ffbe0b`
